Men's Gymnastics Code of Points (unfinished and discontinued)

This was my first attempt at making and Android Application using Android Studio. My goal was to make a Program that...

  • Would allow coaches and gymnasts to access the Men's Gymnastics Code of Points
  • Would allow coaches and gymnasts to view the compulsory routines
  • Be available offline
  • Would be organized enough to access these tools quickly
The problem came during the final steps of the program. I was unable to find a way to view pdf files without internet connection. The code is still available on my github account using the link above.
